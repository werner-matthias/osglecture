# osglecture
=======

LaTeX bundle for generating lecture materials at the Operating Systems Group of the TU Chemnitz.

Currently, the following packages are included:

<!-- * ./osglecture: The **osglecture** basis class to generate different materials (slides, script) from a common source -->
* ./ollm - **OSG LaTeX Lecture Maker**: a self-contained latexmk script to support creation and deploying lecture materials
* ./langselect - **langselect**: bundle to support the generation of different language versions from a common source
* ./tagpax - **tagpax**:  semantic import of tagged PDFs
* ./lttheme - **ltxtalk-theme**: to allow easy and flexible design of themes for ltx-talk
* ./osgdoc - **osgdoc**: support for documentation
