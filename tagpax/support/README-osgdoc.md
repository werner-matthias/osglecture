# osgdoc build support

The generated `osgdoc.cls` and `osgdoc.sty` files are included only so that
`l3build doc` can build the tagpax manual reproducibly. They originate from
the user-supplied osgdoc development source, version 0.0.7, and remain under
the LPPL 1.3c or later. A system-installed osgdoc may be used instead.
